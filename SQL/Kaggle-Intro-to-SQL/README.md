# Kaggle - Intro to SQL

## Project Overview

This project contains the hands-on SQL exercises I completed as part of Kaggle's **Intro to SQL** course. The exercises were performed using **Google BigQuery** and focus on learning fundamental SQL concepts for querying and analyzing data.

## Platform

- Kaggle Learn
- Google BigQuery

## Skills Demonstrated

- SELECT
- FROM
- WHERE
- GROUP BY
- HAVING
- ORDER BY
- AS
- WITH (Common Table Expressions - CTEs)
- INNER JOIN

## Exercises Completed

- Getting Started with SQL and BigQuery
- SELECT, FROM & WHERE
- GROUP BY, HAVING & COUNT
- ORDER BY
- AS & WITH
- Joining Data

## Learning Outcomes

Through these exercises, I learned to:

- Retrieve and filter data from databases.
- Summarize data using aggregate functions.
- Group and sort query results.
- Create readable queries using aliases and CTEs.
- Combine data from multiple tables using JOINs.
- Write efficient SQL queries to answer business questions.
